mysql导入导出

进入 mysql

格式： mysql -u 用户名 -p 用户密码

查看数据库

查看 Mysql 所有的数据库

如果要查看某一具体数据库，格式 use 数据库名 ，for example : use xingxing

导入导出数据为 SQL 文件

例如： mysqldump -u root -p –default-character-set=utf8 xingxing >/usr/local/xingxing.sql
